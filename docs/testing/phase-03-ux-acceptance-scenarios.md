# Phase 03 — UX Acceptance Scenarios

**Status:** Approved Baseline  
**Date:** 23 July 2026  

## Language and Direction

1. Open the application in Arabic RTL.
2. Open the application in English LTR.
3. Switch language without reinstalling.
4. Verify navigation direction.
5. Verify mixed Arabic and English bank descriptions.
6. Verify account numbers remain readable.

## People

1. Add a person.
2. Add a profile image.
3. Replace and remove the image.
4. Add aliases.
5. Search by alias.
6. Display “has with me” and “owes me” separately.
7. Display ILS and USD separately.
8. Generate initials avatar.

## Transactions

1. Record a deposit.
2. Record a withdrawal.
3. Record a loan.
4. Record a repayment.
5. Record personal income.
6. Record personal expense.
7. Preview financial effect.
8. Exit with unsaved changes.
9. Reverse a posted transaction.

## Wallet Transfer

1. Enter 200 ILS.
2. Preview 1.5 ILS commission.
3. Preview 201.5 ILS charged to person.
4. Confirm when funds are sufficient.
5. Confirm debt creation when funds are insufficient.
6. Cancel without financial effect.

## Import

1. Select XLSX.
2. Validate file.
3. Preview new, duplicate, warning, and error rows.
4. Confirm import.
5. Review deterministic suggestion.
6. Review probable duplicate.
7. Finish with import result.

## Reconciliation

1. Enter statement balance.
2. View calculated balance.
3. Understand difference.
4. Resolve missing item.
5. Close at zero difference.

## Budget and Savings

1. Create monthly plan.
2. View salary, spending, remaining budget.
3. View safe daily spend.
4. Create savings goal.
5. Add savings contribution.
6. Close month.

## Statements

1. Generate Arabic WhatsApp statement.
2. Generate English WhatsApp statement.
3. Edit generated text.
4. Copy text.
5. Share text.
6. Preserve unsent draft.

## Accessibility

1. All touch targets are at least 48dp.
2. Screen reader reads important amounts.
3. Color is not the only status signal.
4. Text scaling does not hide critical actions.
5. Keyboard and switch focus order is logical.
